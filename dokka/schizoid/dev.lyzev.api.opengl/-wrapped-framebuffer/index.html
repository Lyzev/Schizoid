WrappedFramebuffer

class WrappedFramebuffer(lod: Int = 0, width: Int = MinecraftClient.getInstance().window.framebufferWidth, height: Int = MinecraftClient.getInstance().window.framebufferHeight, useDepth: Boolean = false, fixedSize: Boolean = false) : SimpleFramebuffer, EventListener

A simple wrapper around the SimpleFramebuffer class. This class is used to create a framebuffer with a size multiplier which uses GlConst.GL_LINEAR as the texture filter.

Parameters

lod

The level of detail of the framebuffer.

useDepth

Whether to use a depth attachment or not.
